## âš¡ Quick Start

### 1. Requirements
Ensure you have **Node.js 18+** and a **Supabase** instance ready.

### 2. Setup
```bash
# Clone the vibe
git clone https://github.com/YUVRAJ-SINGH-3178/VibeSRM.git
```
